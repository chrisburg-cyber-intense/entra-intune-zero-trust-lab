# 10-Minute Interview Demo Script

1. Show architecture diagram (30 seconds)
2. Show Compliance Policy (BitLocker/TPM/Secure Boot) (1 minute)
3. Show BitLocker policy + Recovery Key escrow (1 minute)
4. Show ASR policy (1 minute)
5. Show LAPS policy + local admin password escrow (1 minute)
6. Show Conditional Access policy requiring compliant device (1 minute)
7. Demo Blocked sign-in when noncompliant (2 minutes)
8. Demo Allowed sign-in when compliant (2 minutes)
9. Show sign-in logs proving enforcement (1 minute)
